# 🔄 Como Recarregar o Electron

O frontend agora está rodando! Você precisa recarregar a janela do Electron para carregar o conteúdo.

## Opções:

### Opção 1: Recarregar a Janela (Mais Rápido)

Na janela do Electron, pressione:
- **F5** ou **Ctrl+R** para recarregar a página

Ou clique com o botão direito na janela e selecione "Reload"

### Opção 2: Reiniciar o Electron

1. Feche a janela do Electron
2. Execute novamente:
   ```powershell
   npm run dev:electron-only
   ```

### Opção 3: Usar o Menu do Electron

Se o menu estiver visível:
- View → Reload
- Ou View → Force Reload (Ctrl+Shift+R)
